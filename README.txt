Addition to WP-Members plugin (Chad Butler) that sends an email to the Membership Administrator when members edit specific fields of their profile.

== Description ==
The WPM-Email plugin is an add-on to the WP-Members plugin from Chad Butler.  WPM-Email sends the Membership Administrator an email whenever a member updates his or her membership profile.  The settings page for this plugin allows you to set the Membership Administrator\'s email address and an additional cc email address.  You can select the member profile fields that you are interested in and the plugin will send an email if the member has made any changes to the selected fields. The order of fields is the order set in WP-Members settings.

Tested with WP Members Vers. 3.1.x

== Installation ==
Upload the wpm-email.zip file using the Wordpress Dashboard Upload Plugin page and click on Install;  
or unzip the file in your /wp-content/plugins/ directory.

Activate the plugin through the \'Plugins\' menu in WordPress.

Tested with WP Members V 3.1.x
